# card_balance_checker
Описание репозитория Этот репозиторий содержит скрипт на Python, который позволяет проверять баланс карты, используя API. Скрипт отправляет POST-запрос с номером карты и получает информацию о балансе. В случае успешного ответа, данные обрабатываются с помощью библиотеки BeautifulSoup для извлечения и отображения текущего баланса карты.
